# Day 7 – Claude Usage Strategy
## ABTalks 60-Day Claude AI Mastery Challenge

## What I Learned Today
Different Claude models and effort levels work best
for different tasks. Using the right combination
saves time and gets better results.

## My Profile
- Student: ECE First Year, Coimbatore
- Usage: Daily heavy user
- Activities: Coding, Learning, Content, Career prep

## Model Guide
- Haiku: Quick answers, short captions (5% of time)
- Sonnet: Daily challenge, DSA, resume (90% of time)
- Opus: Career strategy, SIH planning (5% of time)

## Effort Level Guide
- Low: Quick tasks, captions, simple fixes
- Standard: Daily default for most work
- High: Important outputs, career plans
- Max: Rare, only for critical complex tasks

## Biggest Mistakes to Avoid
1. Using Max effort for every task
2. Giving zero context to Claude
3. Accepting first output without iterating
4. Not saving good prompts for reuse

## Key Takeaway
"Right Model. Right Effort. Right Result."
