# Day 10 – Personal Portfolio Website
## ABTalks 60-Day Claude AI Mastery Challenge

## What I Built
A complete personal portfolio website using
Claude AI + HTML + Tailwind CSS + Vanilla JS

## Features
- Hero with typing animation
- About Me with CGPA and status cards
- Animated skill bars + 16 tech tags
- 6 real project cards
- All certifications listed
- Dark/Light mode toggle
- Mobile responsive
- Smooth scroll animations
- Single HTML file — no build step!

## Key Learnings
1. Claude can build full portfolio sites in one prompt
2. Tailwind CDN = no setup, instant styling
3. Single HTML file = deployable anywhere for free
4. Personal branding starts with a portfolio

"Your portfolio is your digital handshake.
Make it count."
