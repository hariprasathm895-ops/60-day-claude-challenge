# Day 5 – Context Engineering
## ABTalks 60-Day Claude AI Mastery Challenge

## What I Learned Today
Context Engineering = giving Claude background information
about YOU before asking a question. More context = better,
more personalized output.

## Prompt A — Without Context
"Create a 30-day learning roadmap."
Output: Generic, could apply to anyone. Not useful.

## Prompt B — With My Context
- Student: ECE First Year, Coimbatore
- Skills: Basic Python, learning DSA
- Goal: IT Placement / AI Engineer
- Time: 2 hours/day
- Level: Beginner
Output: Specific weekly plan, Tamil YouTube resources,
placement-focused, fits my exact schedule.

## Key Comparison
| | Prompt A | Prompt B |
|---|---|---|
| Personalized | No | Yes |
| Would I follow it | No | Yes |
| Placement focus | No | Yes |

## 3 Key Learnings
1. Context = difference between generic and useful
2. More details you give Claude = better output
3. Context Engineering is a superpower for students

## Key Takeaway
"Claude is only as smart as the context you give it."
