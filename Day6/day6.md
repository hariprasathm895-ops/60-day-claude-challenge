# Day 6 – ATS Resume Optimization
## ABTalks 60-Day Claude AI Mastery Challenge

## What I Learned Today
Used Claude as an ATS expert to analyze and optimize
my resume for maximum recruiter and ATS readability.

## ATS Score Improvement
- Previous Score: 58/100
- Optimized Score: 84/100

## Key Changes Made
1. Replaced Career Objective with Professional Summary
2. Added Projects section with real GitHub work
3. Removed declaration line (not ATS-friendly)
4. Strengthened action verbs throughout
5. Added target role keywords for Full Stack Developer

## Key Learnings
- ATS systems scan for keywords — match the job role
- Projects section is critical for fresher resumes
- Professional Summary beats Career Objective always
- Declaration lines waste space and hurt ATS score

## Key Takeaway
"Your resume is a product. Optimize it like one."
